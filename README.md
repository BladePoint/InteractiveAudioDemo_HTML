# InteractiveAudioDemo_HTML
This is a web-based HTML demo of the Interactive Audio Engine implemented in JavaScript. See it in action here:  
https://www.newgrounds.com/portal/view/913019  
https://bladepoint.heliohost.us/demo

Unfortunately, if you download the files and attempt to view the HTML locally off your hard drive, your browser will probably give you CORS errors and it won't work properly. You will have to do one of the following:
1. Disable Cross-Origin Restrictions on your browser, then view the HTML. (not recommended)
2. Upload the files to a web server and view them that way.
3. Set up a local server on your computer, copy the files onto it, and serve them to yourself locally.
